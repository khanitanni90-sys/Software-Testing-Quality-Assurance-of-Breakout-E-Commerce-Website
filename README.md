# Software-Testing-Quality-Assurance-of-Breakout-E-Commerce-Website
This project focuses on the Software Testing &amp; Quality Assurance of the Breakout E-Commerce Website, a Pakistan-based online fashion retail web application. The objective of this project is to ensure the manual testing, automation testing, API testing, and defect tracking to ensure the website’s functionality, reliability, and performance.
The Breakout website allows users to browse products, view product details, add items to the cart, and proceed to checkout. The application follows a **client-server architecture** and uses **backend APIs, cookies, and sessions** to manage user state. This project was conducted as part of the **Software Testing & Quality Assurance course** to gain hands-on experience with real-world QA practices and tools.

##  Tools & Technologies Used
- **Browser:** Google Chrome  
- **Operating System:** Windows 10 / 11  
- **Automation Tools:** Selenium / Playwright  
- **API Testing Tool:** Postman  
- **Bug Tracking Tool:** Jira  
- **API Type:** REST  
- **Data Format:** JSON  
- **Authentication:**Session Cookies  

##  Testing Types Performed
- **Manual Testing** (20–25 test cases)
- **Automation Testing** (8–10 automated scripts)
- **API Testing** (6–8 APIs tested)
- **Defect Tracking & Bug Reporting** (5–7 defects logged)

##  Project Website Details
- **Application Name:** Breakout  
- **Website URL:** https://breakout.com.pk  
- **Application Type:** Web-based E-Commerce Application  
- **Domain:** Online Fashion Retail (Clothing & Accessories)  

##  Conclusion
This project successfully achieved its testing objectives and significantly improved our understanding of **real-world QA processes**. It enhanced our skills in **manual testing, automation, API testing, defect management**, and professional QA documentation.
